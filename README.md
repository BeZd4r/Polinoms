Программа для подсчитывания полинома 1 степени ,состоящего из 6 точек. 
Для подсчета полинома используется МНК (метод наименьших квадратов) и 6 введенных пользователем точек.
Программа имеет возможность сохранять полученную формулу вместе с точками (Файл сохраниния Formala.txt).
Из дополнительных возможностей включена возможность перемещаться по TextBox`ам с помощью клавиши Enter.
